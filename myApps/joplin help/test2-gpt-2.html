What is Joplin?

Joplin is a free, open source note-taking and to-do application, which can handle a large number of notes organized into notebooks. The notes are searchable, can be copied, tagged, and modified either from the applications directly or from your own text editor.

Features

Joplin offers offline-first, markdown-based note-taking, ensuring you can read your notes on your phone or computer anytime. It can sync via multiple services like Nextcloud, Dropbox, and more.

Markdown Support

Joplin uses Markdown for note editing, allowing for quick formatting and easy readability.

Offline Sync

Sync your notes across devices, and keep them available even without an internet connection.

Multiple Platforms

Joplin runs on Windows, macOS, Linux, iOS, Android, and even offers a Web Clipper for browsers.

Installation

Windows

Download the installer from the official website and follow on-screen instructions.

macOS

Use the provided DMG file to drag and drop Joplin into your Applications folder.

Linux

Use the AppImage, or install from repositories if available for your distro.
